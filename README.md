## 이정석의 포토폴리오 사이트

- 본 페이지는 notion을 기반으로 제작되었기 때문에 본 페이지 보다 notion으로 보는게 더 깔끔합니다.
- 본 페이지는 회사의 보안 문제 때문에 notion이 열리지 않을 경우 대용으로 사용할때 쓰기 위해 제작 했습니다.

- 본 페이지는 해당 사이트에서 템플릿을 활용하여 제작 했습니다 (https://html5up.net/hyperspace)
